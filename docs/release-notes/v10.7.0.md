# v10.7.0 - Parameter Whitespace Bug Fix

**Release Date:** 2026-01-11
**NuGet:** [Neatoo.RemoteFactory 10.7.0](https://nuget.org/packages/Neatoo.RemoteFactory/10.7.0)

## Overview

Fixes a bug where multi-line parameter declarations with different indentation patterns caused duplicate Save/TrySave methods to be generated, resulting in CS0111 compilation errors.

## What's New

None

## Breaking Changes

None

## Bug Fixes

- **Fixed duplicate Save method generation**: When factory methods (Insert, Update, Delete) had CancellationToken parameters with different whitespace/indentation patterns, the generator would create duplicate Save/TrySave methods on the factory interface. This was caused by `ToFullString()` including trivia (whitespace) in parameter type strings, which broke the method grouping logic. The fix trims whitespace from parameter types to ensure consistent grouping regardless of source code formatting.

## Commits

- `fix:` Trim whitespace from parameter Type in MethodParameterInfo
