# v0.25.0 - LazyLoad&lt;T&gt; for Deferred Async Loading

**Release Date:** 2026-03-28
**NuGet:** [Neatoo.RemoteFactory 0.25.0](https://nuget.org/packages/Neatoo.RemoteFactory/0.25.0)

## Overview

Adds `LazyLoad<T>`, a general-purpose deferred async loading wrapper with full serialization support. Enables on-demand loading of expensive data across the client/server boundary without eager fetching or extra round-trips.

## What's New

- **`LazyLoad<T>` type** — Wrapper with passive `Value` (never triggers load), explicit `LoadAsync()`, thread-safe concurrent loading, `INotifyPropertyChanged` forwarding, and error tracking (`HasLoadError`, `LoadError`).

- **`ILazyLoadFactory`** — DI-injected factory (singleton) for creating `LazyLoad<T>` instances. Two patterns: `Create<T>(loader)` for deferred loading, `Create<T>(value)` for pre-loaded/eager scenarios.

- **Named-format serialization** — `LazyLoadJsonConverterFactory` serializes as `{"value": ..., "isLoaded": bool}`. Standalone converter in the serializer chain (not a `NeatooJsonConverterFactory` subclass).

- **Ordinal-format serialization** — Generator two-slot encoding: each `LazyLoad<T>` property occupies two consecutive array slots `[value, isLoaded]`. New `IsLazyLoad` and `InnerType` fields on `OrdinalPropertyModel` drive the renderer.

- **Merge pattern** — `ILazyLoadDeserializable.ApplyDeserializedState()` preserves loader delegates across serialization by merging deserialized state into constructor-created instances.

- **Documentation** — Dedicated [LazyLoad](../lazyload.md) page, updated [Serialization](../serialization.md), [Interfaces Reference](../interfaces-reference.md), and [Service Injection](../service-injection.md). RemoteFactory skill updated with LazyLoad triggers and reference guide.

## Usage

```csharp
[Factory]
public partial class ProductWithReviews
{
    public int Id { get; set; }
    public string Name { get; set; } = string.Empty;
    public LazyLoad<string> Reviews { get; set; } = new LazyLoad<string>();

    [Remote, Fetch]
    internal void Fetch(int id,
        [Service] ILazyLoadFactory lazyLoadFactory,
        [Service] IReviewService reviewService)
    {
        Id = id;
        Name = $"Product_{id}";

        // Deferred: set up loader, don't load yet
        Reviews = lazyLoadFactory.Create<string>(async () =>
        {
            return await reviewService.GetReviewsAsync(Id);
        });
    }
}

// Client usage:
var product = await factory.Fetch(42);
// product.Reviews.IsLoaded == false

var reviews = await product.Reviews.LoadAsync();
// product.Reviews.IsLoaded == true
```
